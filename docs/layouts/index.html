Why Blockyard?

Container Isolation

Every Shiny app runs in its own container with network isolation, read-only filesystems, and dropped capabilities.

Automatic Dependencies

Upload a bundle and Blockyard resolves and installs R packages automatically via pak. No renv.lock required.

Cold Start Workers

Containers spawn on demand when the first request arrives. No idle workers wasting CPU and memory.

Auth & RBAC

OIDC sign-in, role-based access, per-app ACLs, and optional OpenBao for credential management.

CLI & REST API

Deploy from your terminal with by deploy or script with the REST API. One config file, one binary.

Self-hosted

Runs anywhere Docker runs. No SaaS dependency, no per-seat fees, your data never leaves your network.
